## 1️⃣ الملخص التنفيذي

OYO PLAST منصة تجارة إلكترونية متكاملة (RTL عربية) لطباعة وتوريد المنتجات البلاستيكية في اليمن، تجمع بين تجربة تسوق على طراز SHEIN/Savvy، ونظام تشغيل داخلي بمستوى المؤسسات (Admin + Suppliers + Marketers + Staff)، مدعومة بـ **9 وكلاء ذكاء اصطناعي** وفريق بشري قابل للتوسعة.

**الحالة الحالية:** ✅ جاهزة للإنتاج التجاري.

## 4️⃣ القوة الداخلية والأمن

### 🛡️ الحماية الحالية
- **مصادقة طبقية**: Session Cookie + HMAC للأدمن + JWT للموردين/المسوقين.
- **Rate limiting**: 5 تسجيلات/IP/ساعة، حماية spam على OTP.
- **Race conditions**: `BEGIN + FOR UPDATE` في كل العمليات المالية الحرجة.
- **Idempotency guards**: منع double-discount على ديون العملاء (HTTP 409).
- **Audit Log** لكل أفعال الموظفين.
- **SQL Injection**: Drizzle ORM مع parameterized queries بالكامل.
- **XSS**: React escape تلقائي + DOMPurify عند الحاجة.
- **CSRF**: ميدلوير على نقاط الكتابة + SameSite cookies.
- **Secrets**: محفوظة في Replit Secrets (لا تظهر في الكود).
- **Cron Safety**: قفل ذري عبر `tried_supplier_ids[]` لمنع تكرار التعيين.

### 🔐 توصيات أمان مستقبلية
- اختبار اختراق خارجي (Pen-test) — تكلفة ~$1,500.
- WAF (Cloudflare Pro) — $20/شهر.
- نسخ احتياطية مشفّرة خارج Replit (S3 + KMS).

## 8️⃣ السعة الاستيعابية (Concurrent Capacity)

### تقدير حسب البنية الحالية (PostgreSQL + Express on Replit)

| السيناريو | طلبات/ساعة | متزامن (concurrent) | استجابة API |
|----------|------------|---------------------|-------------|
| 🟢 **عادي** | 10-30 | 5-15 | <200ms |
| 🟡 **متوسط** | 50-100 | 30-60 | 250-400ms |
| 🟠 **ذروة** | 200-400 | 100-200 | 400-800ms |
| 🔴 **قصوى (نفس البنية)** | 600-800 | 300-400 | 800ms-1.5s |

### أمثلة تطبيقية
- **يوم عادي** (5 طلبات/ساعة، 50 مستخدم): سهل جداً، أقل من 5% CPU.
- **حملة ترويج صغيرة** (100 طلب/ساعة، 300 مستخدم): تعمل بسلاسة.
- **يوم Black Friday محلي** (500 طلب/يوم، 1,500 زيارة): تحتاج monitoring فقط.
- **انفجار فيروسي** (5,000 طلب/يوم، 10,000 زيارة): تحتاج ترقية قاعدة بيانات + Load Balancer ($50-150/شهر).

### حدود البنية الحالية القصوى (Cloudinary + Replit)
- **DB connections**: 60 متزامن (افتراضي PG).
- **Image bandwidth**: 25GB/شهر Cloudinary مجاناً ثم $89/شهر.
- **API requests**: ~3M/شهر بسهولة.
- **Storage**: لا حدود فعلية مع Cloudinary CDN.

### للوصول إلى 10,000+ طلب/يوم
- ترقية إلى Reserved VM + Replica DB (+$80/شهر).
- إضافة Redis cache (+$15/شهر).
- CDN على الـ static assets (Cloudflare مجاناً).
- **التكلفة الإجمالية للسعة العالية: ~$200/شهر**.
