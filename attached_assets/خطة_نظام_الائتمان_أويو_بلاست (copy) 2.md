# 1️⃣ الركن الأول: تصنيف الزبائن

| الفئة | الشروط | السقف الائتماني | مدة السداد | الدفعة المقدمة | خصم الكاش |
|---|---|---|---|---|---|
| 🥇 **VIP** | أكثر من سنة، صفر تأخير | 200,000 ر.ي | 30 يوم | 0% | 3% |
| 🥈 **فضي** | 3-12 شهر، تأخير ≤ 7 أيام | 75,000 ر.ي | 21 يوم | 30% | 5% |
| 🥉 **برونزي** | أقل من 3 شهور (بعد 3 طلبات كاش أولاً) | 25,000 ر.ي | 14 يوم | 50% | 7% |
| ⚫ **محظور** | تأخر أكثر من 30 يوم | 0 | كاش فقط | 100% | لا |

**القاعدة:** كل زبون يبدأ من الأسفل ويصعد بالتدريج حسب انضباطه.

# 2️⃣ الركن الثاني: التسعير الذكي

## القاعدة الذهبية: الخصم لا يخرج من جيبك

السر: السعر المعروض في متجرك يحتوي على هامش مخفي للخصم.

### المعادلة:
```
سعر التكلفة (من المورد): 100%
+ ربحك المستهدف:         25%
+ هامش الخصم النقدي:      7%
─────────────────────────
السعر المعروض:           132%
```

### النتيجة:
| من اشترى | السعر | ربحك |
|---|---|---|
| VIP كاش (-3%) | 128% | **+28%** ✅ |
| فضي كاش (-5%) | 125% | **+25%** ✅ |
| برونزي كاش (-7%) | 122% | **+22%** ✅ |
| أي زبون أجل | 132% | **+32%** ✅✅ |

**أنت تربح في كل الحالات.**

## تصنيف منتجاتك إلى 4 أنواع تسعيرية:

| نوع المنتج | الهامش | الخصم النقدي | متاح أجل؟ |
|---|---|---|---|
| 🟢 **نجم** (هامش > 60%) | عالي | حتى 10% | كل الفئات |
| 🟡 **عادي** (هامش 25-60%) | مريح | 3/5/7% | كل الفئات |
| 🟠 **حساس** (هامش 12-25%) | محدود | 2/3/4% | VIP وفضي فقط |
| 🔴 **هامشي** (هامش < 12%) | ضيق | لا خصم | كاش فقط |

# 3️⃣ الركن الثالث: التحصيل التلقائي

## كيف يعمل قسط واحد (مثال: 7,000 ر.ي يستحق 15 مايو)

```
يوم -3 (قبل الاستحقاق بـ 3 أيام):
   → WhatsApp ودي + إشعار بالتطبيق
   → البطاقة في "حسابي" تتلوّن أصفر

يوم 0 (يوم الاستحقاق):
   → WhatsApp + SMS
   → زر "ادفع الآن" بارز
   → البطاقة برتقالي

يوم +1 إلى +3 (تأخير بسيط):
   → تذكير يومي
   → لا يستطيع طلب جديد

يوم +7 (تأخير أسبوع):
   → 🚫 تجميد الحساب تلقائياً
   → إشعار للأدمن

يوم +30 (تأخير شهر):
   → ⬇️ تخفيض رتبة الزبون آلياً
   → إضافة لقائمة المتعثرين
```

## شاشة "حسابي المالي" للزبون:

```
┌────────────────────────────────────┐
│  💳 حسابي المالي                   │
│  ─────────────────                 │
│  ✨ فئتي: فضي                      │
│  💰 الحد: 75,000                   │
│  📊 المتاح: 45,000                 │
│                                    │
│  ─── الأقساط المستحقة ───          │
│                                    │
│  🟠 7,000 — مستحق 15 مايو          │
│      [ ادفع الآن ]                 │
│                                    │
│  🟡 7,000 — مستحق 22 مايو          │
│      [ ادفع مبكراً (خصم 1%) ]      │
│                                    │
│  📥 كشف حساب PDF                   │
└────────────────────────────────────┘
```

## طرق الدفع (داخل التطبيق، بدون زيارتك):

1. **المحافظ الرقمية** (Cash, Jaib, Floosak) — موجودة مسبقاً
2. **التحويل البنكي** — موجود مسبقاً
3. **رفع صورة الإيصال** → AI يقرأ المبلغ → تأكيد تلقائي

# 4️⃣ الركن الرابع: 7 طبقات حماية

| # | الطبقة | كيف تحميك |
|---|---|---|
| 1 | **سقف ائتماني** | السلة ترفض تلقائياً تجاوز السقف |
| 2 | **دفعة مقدمة** | 30-50% تغطي تكلفتك من المورد |
| 3 | **تجميد تلقائي** | يوم +7 = لا طلبات جديدة |
| 4 | **تخفيض الرتبة** | كل تأخير = مراجعة آلية للفئة |
| 5 | **صندوق احتياطي** | 2% من كل بيع كاش = وسادة للديون |
| 6 | **كفيل شخصي** | للقروض > 100K ر.ي |
| 7 | **سقف التعرّض** | إجمالي الديون لا يتجاوز 30% من إيراداتك الشهرية |

# 📊 مثال شامل: 4 زبائن، نفس الطلب 30,000

### 🥇 VIP (أبو محمد) — يختار التقسيط 21 يوم:
- بدون مقدم
- 10K + 10K + 10K (3 أقساط)
- ربحك الكامل 30%

### 🥈 فضي (أبو سعد) — يختار التقسيط مع المقدم:
- 9,000 الآن (30%) ← يغطي تكلفتك
- 7,000 + 7,000 + 7,000 (خلال 21 يوم)
- رأس مالك يعود فوراً

### 🥉 برونزي (أحمد) — غالباً يختار الكاش (الخصم 7% مغرٍ):
- 27,900 ر.ي كاش
- وفّر 2,100 ر.ي
- ربحك 39% فوراً

### ⚫ محظور (سعيد) — لا يستطيع الطلب:
```
🚫 حسابك مجمّد
سدّد المتأخر (18,000) أولاً
ثم ادفع هذا الطلب كاش (30,000)
المجموع المطلوب: 48,000 ر.ي
```

# 🌟 كلمة أخيرة

أنت سألت: **"من يضمن لي وفاء العميل في الموعد؟"**

الإجابة:
- ❌ لا أحد يضمن لك ذلك بنسبة 100%
- ✅ النظام يقلل مخاطرتك إلى 3-5% بدلاً من 15-20%
- ✅ الـ 95% الباقية تدفع = ربحك يستمر
- ✅ الـ 5% المعدومة تُغطى من الاحتياطي + الدفعة المقدمة

**هذه طبيعة التجارة في كل دول العالم.**
حتى أمازون لديها 1.5% ديون معدومة سنوياً.
الأمان المطلق وهم — الأمان الحقيقي هو الإدارة الذكية.
