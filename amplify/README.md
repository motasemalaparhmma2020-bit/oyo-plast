# دليل مجلد المصادقة (Amplify)

هذا المجلد مُضاف كمكان تحضيري لتهيئة نظام المصادقة عبر AWS Amplify. لا يتضمن أي أسرار أو إعدادات بيئية؛ هو قالب إرشادي للمساعدة في تشغيل "npm create amplify@latest" محليًا أو في بيئة التطوير مثل Replit.

ملاحظة أمان: لا تدفع ملفات تحتوي على مفاتيح أو أسرار (مثل aws-exports.js مع معلومات حساسة) إلى المستودع العام أو الخاص دون تشفير.

ملفات هنا:
- auth-setup.md — خطوات التشغيل والرفع.
- .gitkeep — للحفاظ على وجود المجلد في Git.

إذا أردت، أستطيع لاحقًا توليد إعدادات Amplify جاهزة (بدون مفاتيح) أو إنشاء فرع وPR بدل كتابة مباشرة على main.
